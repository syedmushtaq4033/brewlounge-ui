# 🌟 About The Project

Lounge Café UI is a sophisticated restaurant and café website template focused on immersive visuals, premium typography, and minimal dark-themed layouts.

The design captures a luxurious dining atmosphere while maintaining smooth responsiveness and clean frontend architecture.

Perfect for:

- ☕ Coffee Shops
- 🍽️ Restaurants
- 🥐 Bakery Brands
- 🍷 Lounge Cafés
- 🌙 Luxury Dining Experiences

# 🚀 Local Setup

## Clone repository

```bash
git clone https://github.com/syedmushtaq4033/lounge-cafe-ui.git
```

## Enter project directory

```bash
cd lounge-cafe-ui
```

## Install dependencies

```bash
npm install
```

## Run development server

```bash
npm run dev
```
